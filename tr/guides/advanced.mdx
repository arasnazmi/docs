<Tip>
Bu bölümdeki kaynakları tamamladığınızda C++ bilgisi, Linux'ta C/C++ ile gerçek zamanlı yazılım geliştirme
ve özelleştirilmiş gömülü linux dağıtımları oluşturma konusunda ileri düzey bilgi sahibi olacaksınız.
Bütün kaynakları tamamladığınız takdirde Gemstone projesinin tamamına hakim olarak nasıl geliştirildiğini
öğrenmiş olacaksınız.
</Tip>

### 1.1. Yazılım Geliştirme

C ve C++ konularında uzmanlaşarak ROS2 gibi projelerin çekirdek kodlarını anlamanız için gereklidir.

* [Learncpp.com](https://www.learncpp.com)
* [Implementing Finite State Machine in C++](https://www.aleksandrhovhannisyan.com/blog/implementing-a-finite-state-machine-in-cpp)
* [Multithreading in C++ 1](https://medium.com/codex/c-multithreading-the-simple-way-95aa1f7304a2)
* [Multithreading in C++ 2](https://db.in.tum.de/teaching/ss21/c++praktikum/slides/lecture-10.2.pdf)

### 1.2. Realtime Linux

Linux'un gerçek zamanlı deterministik sistemlerde nasıl kullanılacağını anlamanız için gereklidir.

* [Mastering the FreeRTOS](https://www.freertos.org/media/2018/161204_Mastering_the_FreeRTOS_Real_Time_Kernel-A_Hands-On_Tutorial_Guide.pdf)
* [Emulation FreeRTOS](https://freertos.org/Documentation/02-Kernel/03-Supported-devices/04-Demos/03-Emulation-and-simulation/01-Emulation-and-simulation)
* [Realtime Linux Installation](https://www.ubuntubuzz.com/2012/03/real-time-linux-installation-on-ubuntu.html?m=1)
* [Realtime Programming in Linux 1](https://shuhaowu.com/blogseries.html#rt-linux-programming)
* [Realtime Programming in Linux 2](https://wiki.linuxfoundation.org/realtime/documentation/start)
* [Realtime Programming in Linux 3](https://elinux.org/Real_Time)

### 1.3. Gömülü Linux Dağıtımı Oluşturma

Gemstone'un işletim sisteminin nasıl derlendiğini anlamanız için gereklidir.

* [Embedded Linux on QEMU](https://medium.com/@fprotopapa/embedded-linux-from-scratch-quick-easy-on-qemu-87e761834b51)
* [Linux From Scratch](https://www.linuxfromscratch.org/lfs/view/stable)
* [Buildroot](https://buildroot.org)
* [Debootstrap](https://wiki.debian.org/Debootstrap)
* [Yocto Project](https://docs.yoctoproject.org)
