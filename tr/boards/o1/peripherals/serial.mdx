<Tip>
Bölüm bitiminde Gemstone geliştirme kartında seri port kullanımı hakkında bilgi sahibi olacaksınız.
</Tip>

Seri port, bilgisayarlar ve mikrodenetleyiciler gibi cihazlar arasında veri iletimini sağlayan, 
basit ama etkili bir iletişim yoludur. Bu iletişim genellikle `UART (Universal Asynchronous Receiver-Transmitter)` 
protokolü üzerinden gerçekleşir ve düşük hızda veri iletimi için kullanılır.

Bu haberleşme protokolünün daha iyi anlaşılabilmesi için, bir `USB-TTL UART dönüştürücüsü` aracılığıyla Gemstone geliştirme kartı ile 
bir Linux sistemi arasında nasıl iletişim kurulacağına dair bir örnek üzerinden ilerlenmiştir.

### Seri Port Haberleşmesi

[Hızlı Başlangıç](/tr/quickstart) sayfasındaki [İmaj yazma işlemi](/tr/quickstart#1-2-i̇maj-yazma-işlemi) kısmında anlatıldığı gibi 
Gemstone'a başarılı bir şekilde `Minimal` imaj yazma işlemi tamamlanır. USB-TTL UART dönüştürücü aracılığıyla GPIO14 ve GPIO15 UART1 
pin bağlantıları yapılır. Sonrasında [Gemstone'u Keşfet](/tr/quickstart#2-gemstone’u-keşfet) bölümünde anlatıldığı gibi SSH ile 
Gemstone işletim sistemi komut satırına giriş yapılması gerekmektedir.

* Aşağıdaki komut ile `picocom` isimli minimal terminal programının kurunuz.
```bash
sudo apt install picocom
```

* Seri port açma ve veri gönderme işlemleri için script dosyası oluşturunuz.
```bash
touch serial_test.sh
```

* `nano` editörünü kullanarak `serial_test.sh` dosyasını açınız.
```bash
nano serial_test.sh
```

* Aşağıdaki kodu kopyalayıp `serial_test.sh` dosyasına yapıştırınız.
```bash
#!/bin/bash

SERIAL_PORT="/dev/ttyS3"
BAUD_RATE="115200"

nohup picocom -b $BAUD_RATE $SERIAL_PORT &

while true; do
    echo -n "Teknofest" > $SERIAL_PORT
    sleep 1
done
```

* `CTRL+X`'e bastıktan sonra `Y` tuşuna basıp nano editöründen çıkış yaparak kaydetme işlemini tamamlayınız. 

* `chmod` komutu ile `serial_test.sh` dosyasını çalıştırılabilir hale getiriniz.
```bash
chmod +x serial_test.sh
```

<Note> GPIO14 ve GPIO15 pinlerine takılan USB-TTL UART dönüştürücü cihazı Gemstone işletim sisteminde 
/dev/ttyS3 cihazını oluşturmaktadır.</Note>

* Kullanıcı bilgisayarında [Terminal/Konsol Uygulaması](/tr/quickstart#2-1-1-terminal%2Fkonsol-uygulaması) bölümünde
anlatılan `Tabby.sh` isimli terminal uygulamasında ilgili seri portun açılması gerekmektedir.
