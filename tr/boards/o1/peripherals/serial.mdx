Seri port, bilgisayarlar ve mikrodenetleyiciler gibi cihazlar arasında veri iletimini sağlayan, 
basit ama etkili bir iletişim yoludur. Bu iletişim genellikle **UART (Universal Asynchronous Receiver-Transmitter)** 
protokolü üzerinden gerçekleşir ve düşük hızda veri iletimi için kullanılır.

Bu haberleşme protokolünün daha iyi anlaşılabilmesi için, bir **USB-TTL UART dönüştürücüsü** aracılığıyla Gemstone geliştirme kartı ile 
bir Linux sistemi arasında nasıl iletişim kurulacağına dair bir örnek üzerinden ilerlenmiştir.

### Seri Port Haberleşmesi

[Hızlı Başlangıç](../../../quickstart) sayfasındaki [İmaj yazma işlemi](../../../quickstart#1-2-i̇maj-yazma-işlemi) kısmında anlatıldığı gibi 
Gemstone'a başarılı bir şekilde **Minimal** imaj yazma işlemi tamamlandıktan sonra [Gemstone'u Keşfet](../../../quickstart#2-gemstone’u-keşfet)  
bölümünde anlatıldığı gibi SSH ile Gemstone işletim sistemi komut satırına giriş yapılması gerekmektedir.
**sudo apt update && sudo apt upgrade** komutu ile işletim sistemini güncelleme işlemi bittikten sonra

```bash
sudo apt install picocom
```

komutu ile **picocom** isimli minimal terminal programının kurulması gerekmektedir. Sonrasında aşağıdaki
komut ile ttyS2 seri portunun açılması gerekmektedir.

```bash
sudo picocom --baud 115200 /dev/ttyS2
```

<Note>Neden ttyS2?</Note>

Kullanıcı bilgisayarında [Terminal/Konsol Uygulaması](../../../quickstart#2-1-1-terminal%2Fkonsol-uygulaması) bölümünde
anlatılan **Tabby.sh** isimli terminal uygulamasında ilgili seri portun açılması gerekmektedir.
