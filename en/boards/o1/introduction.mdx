# Overview

<Frame>
  <img src="/images/t3-gem-o1-board.png" alt="T3GEMO1" />
</Frame>

* Quad-core 1.4GHz 64-bit Arm® Cortex®-A53
* Dual-core 800MHz Arm® Cortex® R5 MCU
* 4 TOPS C7x DSP Hardware Accelerator
* 50 GFlop GPU

Features the [Texas Instruments AM67A](https://www.ti.com/product/AM67A) microprocessor with these capabilities.
