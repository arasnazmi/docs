<Tip>
By the end of this section, you will have advanced knowledge in the following areas:

* Mastery of C++ programming language
* Real-time software development with C/C++ in Linux
* Creating customized Embedded Linux distributions
* Full understanding of the Gemstone project
</Tip>

### 1.1. Software Development

Essential for mastering C and C++ to understand core code of projects like ROS2.

1. [Learncpp.com](https://www.learncpp.com)
2. [Implementing Finite State Machine in C++](https://www.aleksandrhovhannisyan.com/blog/implementing-a-finite-state-machine-in-cpp)
3. [Multithreading in C++](https://db.in.tum.de/teaching/ss21/c++praktikum/slides/lecture-10.2.pdf)

### 1.2. Realtime Linux

Essential for understanding how Linux is used in real-time deterministic systems.

5. [Mastering the FreeRTOS](https://www.freertos.org/media/2018/161204_Mastering_the_FreeRTOS_Real_Time_Kernel-A_Hands-On_Tutorial_Guide.pdf)
6. [Emulation FreeRTOS](https://freertos.org/Documentation/02-Kernel/03-Supported-devices/04-Demos/03-Emulation-and-simulation/01-Emulation-and-simulation)
7. [Realtime Linux Installation](https://www.ubuntubuzz.com/2012/03/real-time-linux-installation-on-ubuntu.html?m=1)
8. [Realtime Programming in Linux 1](https://shuhaowu.com/blogseries.html#rt-linux-programming)
9. [Realtime Programming in Linux 2](https://wiki.linuxfoundation.org/realtime/documentation/start)
10. [Realtime Programming in Linux 3](https://elinux.org/Real_Time)

### 1.3. Creating Embedded Linux Distribution

Essential for understanding how Gemstone's operating system is compiled.

14. [Debootstrap](https://wiki.debian.org/Debootstrap)
15. [Yocto Project](https://docs.yoctoproject.org)

### 1.4. Operating System Engineering

These resources are optional and require more intensive and long-term study compared to the above resources. Provided for those who want to learn the answer to "How to write an operating system". Contains detailed information about Kernel and Processor Architectures, the heart of operating systems.

16. [MIT 6.1810: Operating System Engineering](https://pdos.csail.mit.edu/6.1810/2023/overview.html)
17. [MIT 6.828: Operating System Engineering](https://pdos.csail.mit.edu/6.828/2024/xv6.html)
18. [xv6 Operating System Source Codes](https://github.com/mit-pdos/xv6-public)
19. [OSDev Projects](https://wiki.osdev.org/Projects)
