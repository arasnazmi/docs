## Apache Nuttx

Pixhawk
[Flight Controller](https://docs.px4.io/main/en/flight_controller/cubepilot_cube_orangeplus.html)
hardware mostly contains STM32 with Cortex-M architecture. The autopilot named PX4 runs on the real-time operating system called Apache Nuttx and does not have support for Texas AM67a.

In this project, the goal is to run Apache Nuttx and PX4 on Cortex-R5 cores found in T3-GEM-O1 development boards. The current status and tasks to be done are in the table below.

|               |                  |
| :------------ | ---------------: |
| Running NuttX on R5 cores | `Completed` |
| Debug support with JTAG | `Completed` |
| GPIO usage | **In Progress** |  
| Serial port driver support | Volunteer Developer Needed |
| IPC communication between NuttX (Cortex-R5) and Linux (Cortex A57) | Volunteer Developer Needed |
